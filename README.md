# Sistema Predictivo para la Optimización de la Cadena de Frío y Caducidad de Medicamentos 

## Descripción del Proyecto
Este proyecto forma parte de la asignatura **Gestión de Datos para IA**. Su objetivo es desarrollar una solución técnica que gestione datos de almacenamiento de medicamentos y prediga el riesgo de pérdida logística (por ruptura de cadena de frío o caducidad), optimizando la asignación de rutas según la vida útil restante.
